# templates
This repository holds a collection of opinionated templates designed for use with [`semibran/scaffy`][semibran/scaffy].

The following templates are included:

```
app
├── src
│   ├── index.html
│   ├── script.js
│   └── style.scss
├── .gitignore
├── license
├── Makefile
├── package.json
├── readme.md
└── rollup.config.js
```

```
module
├── lib
│   └── {{name}}.js
├── license
├── package.json
└── readme.md
```

```
tool
├── bin
│   ├── help.txt
│   └── {{name}}.js
├── .gitignore
├── license
├── package.json
└── readme.md
```

[semibran/scaffy]: https://github.com/semibran/scaffy
